```markdown
### P1-001: Toast 错误提示不够明显

**报告人**: BetaTester001  
**报告时间**: 2024-01-15 14:30  
**影响范围**: 所有用户的错误体验  
**归属层**: 前端  
**状态**: 已修复

当交易失败时，Toast 错误提示出现在右上角，但颜色不够明显，用户容易忽略。

3. 交易失败
4. 观察错误提示

**预期结果**:
应该有明显的红色错误提示，用户能立即注意到

**实际结果**:
错误提示颜色较淡，容易被忽略

**环境信息**:
- 浏览器: Chrome 120
- MetaMask 版本: 11.5.0
- 账户地址: 0x1234...5678
- 网络: Sepolia

**截图/录屏**:

**修复方案**:
调整 Toast 错误类型的颜色对比度，增加图标大小

- [x] 部署上线
```
